Deployment instructions:
- Once checkouted, run Bootstrap_Win.bat with administrator privileges, in order to create the needed symbolic links.
- Run the /MobileClient/AssetsRepo/AssetProcessor.exe and Deploy some assets to start with.
- Run index.html in Google Chrome with a command line parameter: --allow-file-access-from-files

Usage instructions:
- WorldEditor can be accessed by adding "?WorldEditor" to the URL of the address of the game.
